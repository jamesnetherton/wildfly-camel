### camel-schematron

The https://github.com/apache/camel/blob/camel-{camel-version}/components/camel-schematron/src/main/docs/schematron-component.adoc[schematron,window=_blank]
component utilises http://schematron.com/[Schematron,window=_blank] to make assertions
about the presence or absence of patterns in XML documents.
