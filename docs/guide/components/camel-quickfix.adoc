### camel-quickfix

The https://github.com/apache/camel/blob/camel-{camel-version}/components/camel-quickfix/src/main/docs/quickfix-component.adoc[quickfix,window=_blank] component adapts the http://www.quickfixj.org/[QuickFIX/J,window=_blank] engine for using in Camel. This component uses the standard http://www.fixprotocol.org/[Financial Interchange (FIX),window=_blank] protocol for message transport.
