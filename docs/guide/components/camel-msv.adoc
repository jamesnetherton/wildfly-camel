### camel-msv

The https://github.com/apache/camel/blob/camel-{camel-version}/components/camel-msv/src/main/docs/msv-component.adoc[msv,window=_blank] component performs XML validation of the message body using the MSV Library and any of the supported XML schema languages, such as http://www.w3.org/XML/Schema[XML Schema,window=_blank] or http://relaxng.org/[RelaxNG XML,window=_blank] Syntax.
