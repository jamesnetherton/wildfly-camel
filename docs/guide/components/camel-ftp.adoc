### camel-ftp

The https://github.com/apache/camel/blob/camel-{camel-version}/components/camel-ftp2/src/main/docs/ftp2-component.adoc[ftp,window=_blank] component provides access to remote file systems over the FTP and SFTP protocols.

[source,java,options="nowrap"]
CamelContext camelctx = new DefaultCamelContext();
Endpoint endpoint = camelctx.getEndpoint("ftp://localhost:21000/foo?username=admin&password=admin");
camelctx.createProducerTemplate().sendBodyAndHeader(endpoint, "Hello", "CamelFileName", "test.txt");
