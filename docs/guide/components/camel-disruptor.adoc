### camel-disruptor

The https://github.com/apache/camel/blob/camel-{camel-version}/components/camel-disruptor/src/main/docs/disruptor-component.adoc[Disruptor,window=_blank] component provides asynchronous SEDA behavior much as the standard SEDA Component, but utilizes a https://github.com/LMAX-Exchange/disruptor[Disruptor,window=_blank] instead of a BlockingQueue.
