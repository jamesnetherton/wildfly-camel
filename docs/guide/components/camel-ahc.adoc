### camel-ahc

The https://github.com/apache/camel/blob/camel-{camel-version}/components/camel-ahc/src/main/docs/ahc-component.adoc[ahc,window=_blank] component provides HTTP based endpoints for consuming external HTTP resources (as a client to call external servers using HTTP). The component uses the https://github.com/AsyncHttpClient/async-http-client[Async Http Client,window=_blank] library.
