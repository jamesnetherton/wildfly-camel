### camel-http4

The https://github.com/apache/camel/blob/camel-{camel-version}/components/camel-http4/src/main/docs/http4-component.adoc[http4,window=_blank] component provides HTTP based endpoints for calling external HTTP resources.

[source,java,options="nowrap"]
CamelContext camelctx = new DefaultCamelContext();
camelctx.addRoutes(new RouteBuilder() {
    @Override
    public void configure() throws Exception {
        from("direct:start")
        .to("http4://somehost:8080/simple/myservlet");
    }
});

For secure communication over SSL/TLS, you can use the https4 endpoint:

[source,java,options="nowrap"]
to("https4://somehost:443/simple/myservlet");
