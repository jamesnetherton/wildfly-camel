### camel-snmp

The https://github.com/apache/camel/blob/camel-{camel-version}/components/camel-snmp/src/main/docs/snmp-component.adoc[snmp,window=_blank] component enables you to poll https://en.wikipedia.org/wiki/Simple_Network_Management_Protocol[SNMP,window=_blank] enabled devices or configure an SNMP trap.
